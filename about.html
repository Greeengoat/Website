Data Analytics

My core expertise lies in data analytics. I am well versed in manipulating datasets and applying machine learning algorithms to derive hidden insights from data. I have done several analytics project and one of those involves analysing the social media sentiments of a local popular dessert cafe.

Data Engineering

Manipulating and handling big data can a handful. I have had the experience of handling huge volumes of data and building a layer of API on top of them. With an API, users can query for the necessary data they need, rather than having to comb through these huge data on their system.

Web development

I have built several web applications since I picked up programming two years. I built a university hall management system to help automate many of the rigorous and time consuming tasks with Laravel as the framework. On top of that, I have also built a collaborative ordering platform that helps solve the pain point of collating orders in a group. This was built with MeteorJS. Check them out here!

Data Visualisation

Data visualistion is key to communicating findings to the general audience. I have done some visualistion on mobility data during my time at DataStreamX. One visualisation I did was to narrow down on popular shopping malls and did a origin destination analysis on the shoppers. The shoppers were then clustered based on their origin. With that, I was able to identify key clusters where majority of the shoppers are coming from, which can help shopping malls better target their audience.
